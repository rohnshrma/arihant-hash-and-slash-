CSS Practical Assignment: Box Model and Positions

Objective: Understand and practice the CSS box model, including padding, margin, border, and positions (fixed and relative).

Tasks

Task 1: Basic Box Model Styling

Create a div element with a class name box1.

  • Style it with:
    • Width: 200px
    • Height: 200px
    • Background color: lightblue
    • Padding: 10px
    • Margin: 20px
    • Border: 5px solid black
  • Write a description below the box explaining how the box model affects the total size.
Example HTML: 
<div class="box1">This is box1</div>

Task 2: Nested Box Model

Create a div container with a class name container, and inside it, add two child div elements (child1 and child2).

  • Style the container:
    • Width: 300px
    • Border: 2px solid darkblue
    • Padding: 20px
  • Style the children:
    • Width: 50% of the container
    • Padding: 10px
    • Margin: 5px
    • Border: 3px dashed green

Task 3: Position - Fixed

Create a div with the class name fixedBox.

  • Style it with:
    • Width: 150px
    • Height: 150px
    • Background color: lightcoral
    • Position: fixed
    • Place it at the top-right corner of the viewport.

Task 4: Position - Relative

Create a div with the class name relativeBox.

  • Style it with:
    • Width: 200px
    • Height: 100px
    • Background color: lightgreen
    • Position: relative
    • Shift it 30px to the right and 20px down from its original position.

Task 5: Combined Example

Create a webpage layout with:

  • A header fixed at the top of the viewport (position: fixed).
  • A content area with three sections, each styled with the box model (different padding, margin, and border for each section).
  • A footer with position: relative, placed at the bottom of the page but slightly offset from its normal flow using top.

Bonus Task

Add hover effects:

  • When the user hovers over any div, increase its border-width and change the background-color to a contrasting color.

Submission Requirements

  • Save the HTML and CSS in separate files (index.html and style.css).
  • Ensure proper comments in the CSS to explain the styles applied.
  • The page must be visually appealing and responsive.
